Milestone Q/A

1. How does JavaScript Work?
কিভাবে জাভাস্ক্রিপ্ট কাজ করে?

জাভাস্ক্রিপ্ট হলো ক্লায়েন্ট-সাইড স্ক্রিপ্টিং ল্যাংগুয়েজ এবং সচরাচর ব্যবহার হওয়া স্ক্রিপ্টিং ল্যাংগুয়েজগুলোর মধ্যে সব থেকে বেশি কার্যকরি ও সব থেকে বেশি ব্যবহার হওয়া একটি স্ক্রিপ্টিং ল্যাংগুয়েজ। জাভাস্ক্রিপ্ট হলো ক্লায়েন্ট-সাইড স্ক্রিপ্টিং ল্যাংগুয়েজ অর্থাৎ এটি ক্লায়েন্টের মেশিনে বা ওয়েব ব্রাউজারে রান করে। এটা রান করানোর জন্য প্রত্যেকটা মর্ডাণ ব্রাউজারে একটি করে বিল্ট-ইন জাভাস্ক্রিপ্ট ইঞ্জিন থাকে। যখনই আমরা কোনো জাভাস্ক্রিপ্ট প্রোগ্রাম ব্রাউজারে রান করতে যাই, জাভাস্ক্রিপ্টের কোডগুলো ব্রাউজারের বিল্ট-ইন ইঞ্জিন রিসিভ করে এবং আউটপুট পাওয়ার জন্য সেগুলোকে রান করে। একটি স্ট্যান্ডার্ড জাভাস্ক্রিপ্ট ইঞ্জিনে, আমাদের লেখা কোডগুলো বিভিন্ন ধাপের মধ্য দিয়ে যায়। যেমনঃ Parser, AST (Abstract Syntax Tree) etc.

2. What does JavaScript event loop do?
জাভাস্ক্রিপ্ট ইভেন্ট লুপ কী করে?

জাভাস্ক্রিপ্ট সিঙ্গেল-থ্রেডেড সিনক্রোনাস প্রোগ্রামিং ল্যাংগুয়েজ অর্থাৎ যেখানে জাভাস্ক্রিপ্ট কোড রান করা হয় সেখানে একই সময় শুধুমাত্র এক লাইন কোড রান করে। এখন সরাসরি ইভেন্ট লুপ কী সেটা জানতে গেলে আমাদের আরো কিছু বিষয়ের সাথে পরিচিত হতে হবে। আমাদের প্রোগ্রামের ডাটা Heap memory তে এসে জমা হয়। মেমোরি স্ট্যাক আকারে বরাদ্দ যেটা সাধারণত ফাংশনের জন্য ব্যবহৃত হয়। ফাংশন কল স্ট্যাক যেটা আসলে অন্যান্য ফাংশনগুলো কখন কোনটা রান হবে সেটা ট্রেক রাখে। ইভেন্ট লুপ স্ট্যাক এর বাইরের জিনিসগুলোকে টেনে আনে এবং যখনই ফাংশন স্ট্যাক খালি হয়ে যায় তখন এটিকে ফাংশন এক্সিকিউশন স্ট্যাকের উপর রাখে। ইভেন্ট লুপের মাধ্যমে জাভাস্ক্রিপ্ট আমাদের একটা মাল্টিথ্রেড ফিল দেয় যদিও আমরা সবাই জানি জাভাস্ক্রিপ্ট একটা সিঙ্গেল-থ্রেডেড প্রোগ্রামিং ল্যাংগুয়েজ।

3. What is the difference Between Local Storage vs Session Storage?
লোকাল স্টোরেজ এবং সেশন স্টোরেজের মধ্যে পার্থক্য কী?

HTTP Protocol হলো সার্ভার এবং ক্লায়েন্টের মধ্যে Communication এর জন্য সবচেয়ে গুরুত্বপূর্ণ প্রোটোকলগুলির মধ্যে একটি। কিন্তু HTTP Protocol এর প্রধান অসুবিধা হল এটি স্টেটলেস প্রোটোকল অর্থাৎ সার্ভার এবং ক্লায়েন্টের মধ্যে Communication এর তথ্য এটি ট্র্যাক করতে পারে না। তাই এই সমস্যাটির সমাধান করার জন্য, দরকারী তথ্য ট্র্যাক করার তিনটি উপায় রয়েছে এবং সেগুলো হল লোকাল স্টোরেজ, সেশন স্টোরেজ এবং কুকিজ।
Local Storage
Local Storage হল একটি Property যেটা জাভাস্ক্রিপ্ট সাইট অথবা অ্যাপসগুলোকে মেয়াদহীন কী-ভ্যালু জোড়া সংরক্ষণ করতে দেয়। এর মানে ব্রাউজার উইন্ডো বন্ধ হওয়ার পরেও ব্রাউজারে সংরক্ষিত ডেটা বজায় থাকবে।
Session Storage
Session Storage অবজেক্ট শুধুমাত্র একটি সেশনের জন্য ডেটা সঞ্চয় করে। অর্থাৎ ব্রাউজার উইন্ডো বন্ধ হওয়ার পরে ব্রাউজারে সংরক্ষিত ডেটা রিমুভ হয়ে যাবে।
